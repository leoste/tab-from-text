# tab-from-text

Provides a relatively easy way to generate repetitive guitar tabs from code, such as rhythm tabs for punk rock songs.

I wrote this cause clicking and copy-pasting in existing tab editor websites or programs was too tedious to me, and even hand-drawing was a bit annoying. So I decided to write something very niche, a way to generate guitar tabs from code.

It's very limited, there are no pause symbols for example. The system is built around 1/8th notes, though it can support 1/16 derivatives. It's only useful for writing existing songs from your head into tabs, not composing.
